## 📖 About

A production-architected e-commerce storefront built entirely with **vanilla JavaScript** — no React, no Vue, no frameworks. This project proves deep understanding of core JavaScript principles: DOM manipulation, event delegation, async/await, localStorage, and modular code organization.

Built before learning React — the foundation that makes framework knowledge meaningful.

## 📋 Pages & Modules

| Page | File | Purpose |
|------|------|---------|
| Homepage | `index.html` | Product listings with filters |
| Product Detail | `product.html` | Single product view with reviews |
| Cart | `cart.html` | Cart management |
| Checkout | `checkout.html` | Order form and summary |
| Order Confirmation | `order-confirmation.html` | Success page |
| Login | `login.html` | User authentication |
| Register | `register.html` | New account creation |
| Profile | `profile.html` | User account page |
| Advanced Search | `advanced-search.html` | Filtered product search |
| Admin | `manage-products.html` | Product management dashboard |

## ⚡ JavaScript Modules

| Module | Responsibility |
|--------|---------------|
| `auth.js` | Login, register, session management |
| `cart.js` | Cart state, add/remove/update |
| `checkout.js` | Order processing logic |
| `product.js` | Product data and rendering |
| `reviews.js` | User reviews system |
| `search.js` / `advanced-search.js` | Search and filter engine |
| `stock.js` | Inventory management |
| `utils.js` | Shared helper functions |
